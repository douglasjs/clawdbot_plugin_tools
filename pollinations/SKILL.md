# Pollinations.ai Image Generation

Generate free AI images using the Pollinations.ai API.

## Usage

Run the generator script with a text description.

```bash
node /home/node/clawd/skills/pollinations/generate.cjs "A futuristic city with flying cars, cyberpunk style"
```

## Output

The script will:
1. Output `MEDIA: <url>` for preview.
2. Save the image to `/home/node/clawd/persistent/small_pciture_share_app/uploads/`.
3. Output `LINK: https://cortana-pics.dextlas.com/`.
